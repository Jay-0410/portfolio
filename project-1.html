Live Link

Project Overview

This project is a full-stack clone of Airbnb , designed to replicate key features of the platform. It includes essential functionalities such as user authentication, allowing users to securely register, log in, and manage their accounts.

The project also implements CRUD (Create, Read, Update, Delete) operations, enabling authorized user to add, update, or remove listings. Additionally, the reviews feature allows users to leave feedback on listings, fostering community engagement and trust.

One of the standout features is the integration of current location tracking, which displays the exact location of a listing on the map, providing users with real-time information on where properties are located.
